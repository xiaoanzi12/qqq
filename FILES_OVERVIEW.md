## 📊 文件详细说明

### main.c

**文件类型**：C源代码文件

**主要内容**：
- ✅ 完整的LED流水灯程序
- ✅ 使用STC15.h头文件
- ✅ 包含详细的中文注释
- ✅ 提供两种实现方法（查表法和移位法）

**何时使用**：
- 默认情况下使用此文件
- 有STC15.h头文件时
- 需要使用STC15特殊功能时

**关键函数**：
```c
Delay_ms()      // 延迟函数
System_Init()   // 系统初始化
main()          // 主函数
```

---

### main_compatible.c

**文件类型**：C源代码文件

**主要内容**：
- ✅ 与main.c功能相同
- ✅ 使用标准reg52.h头文件
- ✅ 手动定义必要的STC15寄存器
- ✅ 更好的兼容性

**何时使用**：
- Keil找不到STC15.h时
- 需要更好兼容性时
- 跨平台开发时

**使用方法**：
```
1. 在Keil项目中移除main.c
2. 添加main_compatible.c
3. 其他配置不变
```

---

### STC15.h

**文件类型**：C头文件

**主要内容**：
- ✅ 标准8051寄存器定义
- ✅ STC15扩展寄存器定义
- ✅ 端口模式配置寄存器
- ✅ 位定义（sbit）

**何时使用**：
- 使用main.c时必需
- 开发其他STC15项目时可复用

**重要寄存器**：
```c
P0-P7           // 端口寄存器
P0M0-P7M0       // 端口模式配置0
P0M1-P7M1       // 端口模式配置1
TCON, IE, IP    // 标准8051寄存器
```
